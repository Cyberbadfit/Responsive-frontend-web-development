## 📌 Overview

**NEON CONNECT 4** is a modern recreation of the classic Connect Four game with a cyberpunk-inspired interface.

Challenge the AI across three difficulty levels while enjoying smooth gameplay, neon lighting effects, animated game pieces, and an immersive futuristic design.

Perfect for beginners learning JavaScript game development as well as anyone looking for a stylish browser game.

## 🎯 How to Play

1. Select a difficulty level:
   - Easy
   - Medium
   - Hard

2. Click any column to drop your disc.

3. Connect **4 discs** horizontally, vertically, or diagonally before the AI.

4. Win the game or challenge yourself again!

---

## 🧠 AI Difficulty

### 🟢 Easy

- Mostly random moves
- Beginner friendly

### 🟣 Medium

- Smarter positioning
- Occasionally blocks the player
- Balanced gameplay

### 🔴 Hard

- Strategic move evaluation
- Prioritizes winning moves
- Blocks player attacks
- Prefers center control
