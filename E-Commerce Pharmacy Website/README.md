# 💊 Unity Medics - Online Pharmacy Website

A modern, responsive **E-Commerce Pharmacy Website** built using **HTML, CSS, and JavaScript**. 
This project provides users with a clean and user-friendly interface to browse medicines, search products,
manage their shopping cart, authenticate users, and simulate online medicine purchasing.

---

## 📌 Project Overview

Unity Medics is a front-end online pharmacy website designed to provide a seamless shopping experience for customers looking to purchase medicines online.

The website includes product browsing, search functionality, login/signup system, shopping cart management, contact support, and responsive design for all devices.
