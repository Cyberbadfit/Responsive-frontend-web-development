# ✨ FlowTask - Modern To-Do List Web App

FlowTask is a modern and responsive **To-Do List Application** built with **HTML, CSS, and JavaScript**. It helps users organize daily tasks with a beautiful interface, progress tracking, task filtering, and local storage support.

---

## 📌 Overview

FlowTask is designed to improve productivity with a clean and aesthetic user interface. Users can create, edit, complete, filter, and delete tasks while their progress is automatically tracked and saved in the browser.

---

## 🚀 Features

### ✅ Task Management
- Add new tasks
- Edit tasks inline
- Mark tasks as completed
- Delete tasks
- Automatic task timestamp

### 📊 Progress Tracker
- Live progress percentage
- Animated progress bar
- Completion statistics

### 🔍 Task Filters
- View All Tasks
- View Active Tasks
- View Completed Tasks

### 💾 Local Storage
- Saves tasks automatically
- Data remains after refreshing the browser
- No database required

### 🎨 Modern UI
- Glassmorphism design
- Soft gradient background
- Floating animated blobs
- Smooth animations
- Responsive layout
